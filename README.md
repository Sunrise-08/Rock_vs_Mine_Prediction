# Rock vs Mine Prediction

A machine learning project that predicts whether an object detected by sonar signals is a **Rock** or a **Mine** using classification algorithms.

## Project Overview

This project uses sonar signal data to train a machine learning model capable of distinguishing between rocks and underwater mines.

The notebook includes:
- Data preprocessing
- Feature analysis
- Model training
- Prediction system
- Accuracy evaluation

---

## Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn

---

## Dataset

The dataset contains sonar signal readings bounced off different surfaces.

- `R` → Rock
- `M` → Mine

---

## Machine Learning Workflow

1. Load sonar dataset
2. Split training and testing data
3. Train classification model
4. Evaluate model accuracy
5. Predict rock or mine from input data
